# Sessao 6 — Nomeacao + Canvas + Visual

**Fase PONTE:** N — NOMEAR
**Semana:** 6
**Duracao:** 75-90 min

---

## Objectivo da Sessao

Dar nome definitivo ao Signature Framework, preencher o Framework Canvas completo e criar uma representacao visual que possa ser usada imediatamente em apresentacoes, propostas e conteudo. No final desta sessao, o framework esta born — tem nome, estrutura, logica e visual.

---

## Preparacao do Mentorado

Antes da sessao o mentorado deve:
- Trazer a lista de 10 possiveis nomes
- Ter as 1-2 frases por fase ja escritas
- Ter pensado em possiveis acronimos
- Opcionalmente: ter pesquisado como frameworks de referencia foram nomeados

---

## Estrutura da Sessao

| Bloco | Duracao | Descricao |

## Exercicios ao Vivo

### Exercicio 1 — Avaliacao de Nomes com 3 Criterios

Para cada candidato da lista de 10, pontuar de 1-5 em cada criterio:

| Criterio | Definicao | Perguntas-guia |
|----------|-----------|----------------|
| **Memoravel** | Fica na cabeca apos ouvir uma vez | E curto? Tem ritmo? E facil de pronunciar? Tem sonoridade propria? |
| **Ownavel** | Pode ser associado exclusivamente a este profissional | Existe ja algures? E suficientemente especifico? Da para registar? |
| **Significativo** | Comunica o resultado ou a logica do metodo | Um estranho entende do que se trata? Evoca a transformacao prometida? |

Tabela de avaliacao:

| Nome | Memoravel (1-5) | Ownavel (1-5) | Significativo (1-5) | Total |
|------|----------------|--------------|---------------------|-------|
| Nome 1 | | | | |
| Nome 2 | | | | |
| ... | | | | |

O nome com maior pontuacao e o ponto de partida. Se nenhum satisfaz os 3 criterios com pontuacao >=4 em cada, fazer mini-brainstorm na sessao combinando elementos dos melhores candidatos.

**Verificar acronimo** (se aplicavel): o acronimo forma uma palavra com significado? Esse significado reforça a promessa?

### Exercicio 2 — Framework Canvas Completo

Preencher todos os campos:

```

================
Nome: _______________________________________________

_______________________________________________

_______________________________________________

_______________________________________________

MECANISMO UNICO (o que e diferente de outros metodos):
_______________________________________________

PROVA SOCIAL (uma historia de sucesso que demonstra):
_______________________________________________
```

### Exercicio 3 — Representacao Visual

Criar um diagrama simples (pode ser a mao, depois digitalizar):

Opcoes por tipo de framework:
- **Sequencial:** setas lineares da esquerda para a direita, cada fase num bloco
- **Ciclico:** circulo com fases nos pontos cardinais, setas a indicar fluxo
- **Hierarquico:** piramide com niveis, base mais larga em baixo
- **Matricial:** 2x2 ou 3x3 com eixos claramente nomeados
- **Principios:** constelacao central com principios irradiando

Elementos do visual:
- Nome do framework em destaque
- Nome de cada fase/elemento
- Setas ou conectores que mostram a logica de progressao
- Transformacao (De → Para) visivel

## Notas para o Mentor

**Armadilhas comuns:**
- Mentorado que quer esperar para ter o nome "perfeito" — reforcar que o nome pode evoluir, mas precisam de um para avançar. Versao 1 nao e versao final.
- Canvas preenchido com linguagem demasiado tecnica ou jargao — a linguagem deve ser a do cliente, nao do especialista. Testar: "Um cliente entenderia isto?"
- Visual demasiado complexo para ser explicado em 2 minutos — menos e mais. Se precisar de mais de 60 segundos para explicar o visual, e demasiado.
- Mentorado que quer incluir TUDO no framework — ajudar a perceber que a versao publica e a ponta do icebergue, nao o icebergue inteiro.

**Tips de facilitacao:**
- Este e provavelmente o momento mais emocionante do programa — o framework tem nome pela primeira vez. Criar espaco para esse momento.
- Dizer em voz alta: "[Nome do mentorado], apresento-te o [Nome do Framework]." — simbólico mas poderoso.
- Se o mentorado fotografar o canvas no final, e uma prova concreta do trabalho realizado em 6 sessoes
- O Teste dos 3 Niveis no homework e critico — os proximos passos dependem de como o mercado reage
- Antes de fechar: "Em 6 sessoes passaste de intuicao dispersa para um framework com nome, estrutura e visual. Isso e raro e real."
