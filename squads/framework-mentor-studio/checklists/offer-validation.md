# Offer Validation Checklist

## Purpose
Validate that a premium mentoring offer is compelling, priced correctly, and ready for market.

## Checklist

### Value Equation (must pass all)
- [ ] Dream Outcome clearly defined (what the client gets)
- [ ] Perceived Likelihood supported by proof (testimonials, credentials, guarantee)
- [ ] Time Delay minimized (quick wins in first session/week)
- [ ] Effort & Sacrifice minimized (what they DON'T have to do is clear)
- [ ] Value Equation score >= 4.0

### Offer Stack (must pass 5/6)
- [ ] Core offer is clearly defined (format, duration, frequency)
- [ ] At least 2 bonuses that enhance the core
- [ ] Guarantee that reduces risk for the buyer
- [ ] Offer has a compelling name
- [ ] Clear differentiation from "just coaching" or "just consulting"
- [ ] Offer solves a specific problem (not vague "growth")

### Pricing (must pass all)
- [ ] 3 pricing tiers (Good / Better / Best)
- [ ] Pricing is value-based (not time-based)
- [ ] Each tier has clear differentiation
- [ ] Price anchoring: highest tier makes middle tier look reasonable
- [ ] ROI is articulable ("This is worth R$X because...")

### Proposal (must pass 4/5)
- [ ] Proposal is 2.5 pages or less
- [ ] Includes: Situation, Objectives, Metrics, Value, Methodology, Options, Terms
- [ ] Focuses on outcomes, not deliverables
- [ ] Written in client's language (not jargon)
- [ ] Has clear next step / call to action

### Market Fit (must pass 3/4)
- [ ] Avatar is specific enough to find (not "everyone")
- [ ] Avatar has budget for this price point
- [ ] Problem is urgent enough to pay premium
- [ ] There's a way to reach this avatar (warm network, content, referrals)

**PASS:** >= 22/25

**FAIL:** < 18/25

## Red Flags (automatic FAIL)
- Pricing is hourly-based
- No guarantee at all
- Value Equation < 3.0
- Avatar is "anyone who wants to grow"
- Single price point (no tiers)
